# Data_Visualization_Project_1
Data Visualization project 1
This is my first Data Visualization project on Github.
This project is an Exploratory Data Analysis (EDA) of the ais (Australian Institute of Sport) dataset. It focuses on summarizing the physical and blood-related characteristics of elite athletes across various sports.

## Overview

1. Data Inspection
Source: It uses the DAAG library to load data on 202 athletes.
Overview: It uses str() and summary() to look at variables like height, weight, and blood markers (like hemoglobin).

2. Univariate Analysis (Focus on Hemoglobin)
The code investigates the distribution of Hemoglobin (hg) levels using three different methods:
Boxplot: To identify the median and potential outliers.
Histogram: To see the frequency distribution.
Density Plot: To see a smoothed version of the distribution (the "shape" of the data).

3. Categorical Analysis (Sports & Sex)
The code explores how the participants are distributed across different categories:
Frequency Tables: Counts how many athletes are in each sport.
Bar & Pie Charts: Visualizes which sports are most common in the dataset.
Comparative Bar Plots: Compares the number of Male vs. Female participants within each specific sport (e.g., seeing if there are more men in rowing vs. women in netball).

4. Bivariate Analysis (Physical Traits)
Scatter Plot: It finishes by plotting Weight vs. Height to visualize the correlation between an athlete's size and mass.
In short: This is a foundation-level data science project designed to understand the demographics and physiological profiles of professional athletes.
